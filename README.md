# Projeto: SmartMercearia

Sistema de gestão para minimercados desenvolvido como projeto da disciplina de Desenvolvimento de Sistemas Corporativos.

# Sobre

O sistema SmartMercearia tem como objetivo automatizar processos de um minimercado, oferecendo funcionalidades para gerenciamento de estoque, vendas, clientes, fiados, caixa e geração de relatórios.

Esse sistema representa a união de vários conceitos abordados nas disciplinas da graduação em **Sistemas de Informação**, as quais são:

- Algoritmos;
- Estruturas de Dados;
- Banco de Dados;
- Programação Orientada a Objetos (POO);
- Engenharia de Software;
- Análise e Projeto Orientado a Objetos (APOO);
- Desenvolvimento de Sistemas Corporativos.

# Funcionalidades

- Gestão de estoque por lote.
- Registro e gerenciamento de vendas.
- Cadastro de clientes e controle de fiados.
- Controle de caixa e movimentações financeiras.
- Relatórios gerenciais.
- Controle de acesso por perfis.

# Tecnologias utilizadas

- Java
- MySQL / MariaDB
- Linux
- Git e Github
